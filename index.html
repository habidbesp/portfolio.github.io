Astroshare

Developed in the 8th week of the bootcamp at IronHack (Campus Berlin) in October 2020, this group project helped me get a better focus on teamwork by experiencing git and github as a group for the first time for a single project. Also in this project I had my first approach to the Back-end, creating an authentication as well as creating routes to facilitate the user to save images in their own profile.

Technologies:
  • React,
  • Express,
  • Mongoose (data-base),
  • NASA API APOD (Astronomical Picture of the Day)

About Me
